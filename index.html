My Projects

Website Redesign for Local DV Shelter

I collaborated with a local domestic violence shelter to redesign their outdated website. Over several months of meetings and revisions, I developed a new site from scratch, retaining key design elements from the original. The new website includes:

  • Donation Payments: A seamless system for collecting donations online.
  • Event Galleries: Showcasing photos from their activities and events.
  • Social Media Integration: Direct links to their profiles.
  • Program Information: Detailed sections outlining their initiatives.
  • Contact & Volunteer Forms: Fully digitized and functional forms for inquiries and volunteer submissions.

This project is an ongoing effort to ensure the site meets their evolving needs.

Technologies: HTML, CSS, PHP, JavaScript

Food Truck Web Application

This project was created for a class assignment, with the objective of designing a webpage for a fictional food truck. The website featured several key functionalities:

  • Order Management: Customers could place orders, receive order numbers, and track their orders.
  • Google Location Services: Integrated to display the truck's daily locations.
  • Order Ahead: Customers had the ability to pre-order food.
  • Admin Features: Truck owners could log in to update inventory and cancel unclaimed orders.
  • Database Integration: A local database was used to manage orders and track inventory.

This project demonstrated both front-end and back-end development skills, highlighting the seamless integration of functionality with user needs.

Technologies: HTML, CSS, PHP

Raspberry Pi Security Camera

This project was developed in collaboration with a professor, focusing on creating a Raspberry Pi-based security camera. Using a Raspberry Pi 3B, Node.js, and a cell phone, I built a system capable of:

  • Recording video at 720p at 30fps
  • Taking motion-detected snapshots and short clips (less than 10 seconds)
  • Operating with night vision

Technologies: Raspberry Pi 3B, JavaScript, Node-RED

Autonomous Lawn Mower Prototype

This project was by far the most complex I’ve worked on. It started with a lawn mower unit that was rigged for remote control. Using a Raspberry Pi 4B mounted on the mower and connected to the motor controller, I installed the appropriate drivers to enable functionality. Additionally, I mounted peripherals including a LiDAR unit, a 1080p webcam, and ultrasonic sensors, allowing the system to perform object detection.

Everything was programmed using Python 3 and could be accessed remotely via SSH on a Linux Ubuntu machine. The last major update I completed before my contract expired was enabling the drone camera to recognize colors and building a prototype convolutional neural network (CNN) in Jupyter, which allowed the system to identify road cones.

Technologies: Linux, SSH, Python3, Raspberry Pi 4B
